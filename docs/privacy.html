optim-agent

Privacy Policy

Last updated: July 15, 2026

optim-agent does not collect, store, or transmit personal data, telemetry, analytics, or usage logs.

What optim-agent does

optim-agent is an open-source Python package and Claude Code plugin for optimizing configurable systems against measurable objectives. The plugin runs locally inside Claude Code and provides workflow instructions for experiments.

Data collection

optim-agent does not operate a hosted backend and does not receive user data. It does not include telemetry, analytics, tracking pixels, advertising code, or remote logging.

Local project data

When a user asks Claude Code to use optim-agent, Claude Code may read local project files or run local commands as part of the user-directed optimization workflow. Any study files or experiment artifacts are created in the user's local workspace unless the user explicitly stores them elsewhere.

Third-party agent services

optim-agent can be used with Claude Code, Codex, OpenCode, or other agent CLIs. Any data sent to those tools is handled by those services according to their own privacy policies and account settings. optim-agent itself does not proxy, inspect, or store that data.

Contact

For privacy questions, open an issue at github.com/Optim-Agent/optim-agent.
